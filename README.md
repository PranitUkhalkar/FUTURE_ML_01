# AI Sales Forecasting Internship Project

This repository contains my work for **Task 1 of the AI Sales Forecasting Internship**.
The project focuses on forecasting retail sales using time-series models
and presenting business insights through an interactive Power BI dashboard.

## Project Highlights
- Sales forecasting using historical data
- Actual vs Forecast sales comparison
- Holiday and seasonal impact analysis
- Business-focused Power BI dashboard

## Tools & Technologies
- Python (Pandas, Prophet)
- Power BI
- Time Series Forecasting
- CSV / Excel Data

## Repository Structure
- `Sales_Dashboard/` – Power BI dashboard and detailed documentation
- `Notebooks/` – Forecasting model and data preparation
- `Data/` – Sales datasets used for analysis
