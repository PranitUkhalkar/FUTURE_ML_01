# Sales Forecasting Dashboard – Power BI

## Overview
This folder contains the Power BI dashboard created for **Task 1 of the AI Sales Forecasting Internship**.
The dashboard compares actual sales with forecasted sales and provides actionable business insights.

## Objectives
- Analyze Actual vs Forecast Sales
- Measure Forecast Accuracy
- Track Sales Growth Rate
- Evaluate Holiday Impact on Sales
- Understand Seasonal Trends
- Compare Performance Across Store Types

## Tools Used
- Power BI
- DAX
- Time Series Forecasting Output
- CSV / Excel Data

## Key Features
- Actual vs Forecast Sales Line Chart
- Confidence Interval (Upper & Lower Bounds)
- KPI Cards:
  - Total Actual Sales
  - Total Forecast Sales
  - Forecast Accuracy
  - Forecast Deviation
  - Sales Growth Rate
- Holiday vs Non-Holiday Sales Analysis
- Seasonal Sales Distribution
- Store Type Performance Analysis

## Files in This Folder
- `Sales_Forecasting_Task1.ipynb` – Model & data preparation
- `Power BI Dashboard (.pbix)` – Interactive visualization (local file)

## Notes
- Open the Power BI file using **Power BI Desktop**
